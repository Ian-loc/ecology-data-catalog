# Catálogo de Dados Ambientais

Catálogo público e pesquisável de fontes de dados para ecologia, biodiversidade, clima, carbono, solo, vegetação e temas relacionados.

## Objetivo

Facilitar a descoberta de fontes relevantes para ensino, extensão e pesquisa científica, priorizando dados de acesso público e gratuito.

## Conteúdo

- site estático com busca e filtros;
- catálogo disponível em CSV e JSON;
- links para as fontes oficiais;
- indicação de cobertura, formatos, acesso, licença e API.

## Limite de uso

O catálogo é um ponto de partida. Antes de utilizar um dataset, confira versão, licença, metodologia, resolução, cobertura, incerteza e adequação à pergunta científica.
